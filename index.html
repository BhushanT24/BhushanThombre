My Projects

INDIAN SIGN LANGUAGE RECOGNITION SYSTEM

The Indian Sign Language Recognition System is a project aimed at developing a robust and accurate system capable of recognizing and interpreting Indian Sign Language (ISL) gestures. This project holds significant importance in enhancing communication accessibility for the hearing impaired community in India. By leveraging computer vision and machine learning techniques, the system will be able to interpret gestures made by individuals using ISL and translate them into text or spoken language.

POINT OF SALES APPLICATION

The point of sales application is used to generate invoices for products purchased in a store. It utilizes React as the frontend framework, Redux for state management, and Node.js with Express as the backend technology. MongoDB is used as the database for storing data.

INTERIOR DESIGN WEBSITE

Designed and deployed a website for an interior design company, utilizing Amazon Web Services to ensure high performance and reliability.
